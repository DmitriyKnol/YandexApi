# YandexApi
Тетирование API Яндекс диска.
Проверка возможности создания новой папки

Запуск:
<p> -mvn clean test </p>
<br>Запуск Allure отчета:
<p> -mvn clean verify allure:serve </p>
<br>Задействованы инструменты:
<ul>
<li>Java (JDK17)</li>
<li>Maven</li>
<li>Junit4</li>
<li>RestAssured</li>
<li>Allure</li>

</ul>
